# Black-Box-Merging

This repository contains the open-source code and appendix materials for Black-Box Model Merging.

## Contents

- `BMM_code/`: standalone implementation, API, frontend, presets, datasets, and curated adapter pool.
- `Appendix/`: LaTeX appendix material and supporting result tables for the paper.

For setup and usage, see [`BMM_code/README.md`](BMM_code/README.md).

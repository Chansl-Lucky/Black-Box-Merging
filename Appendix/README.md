# Appendix Materials for Black-box Model Merging

This directory contains the supplementary appendix PDF referenced from the anonymous submission.

## File

- `post-bibliography-appendix.pdf`: the three post-bibliography appendix pages with the full result tables and supplementary analysis.

## What is included

The PDF contains the materials cited by the paper when space-limited average tables are shown in the main text:

- baseline hyperparameter grid-search summary,
- oracle top-K source-adapter selections,
- in-domain robustness table with distractor tasks,
- detailed baseline descriptions,
- Qwen full-result tables,
- full ablation breakdown.

## Mapping from main-text references

- `Table 4` in the main paper points to the full ablation breakdown in this PDF.
- `Table 5` in the main paper points to the full Qwen ID/OOD tables in this PDF.
- The oracle top-K discussion and the robustness discussion both point to tables in this PDF.

## Note

The anonymous paper keeps compact summary tables in the submission PDF and moves the larger, post-bibliography tables here for easy reviewer access.
